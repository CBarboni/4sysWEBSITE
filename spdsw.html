SPDSW

Ambiente de programação de CLP

O SPDSW é um ambiente de software desenvolvido para a configuração, programação, depuração, documentação de programas e supervisão dos CLPs da HI Tecnologia. Integrado em um único aplicativo, o SPDSW disponibiliza a linguagem de programação LADDER, e oferece todos os recursos encontrados nos pacotes de programação para CLPs, incluindo depuração, monitoramento on-line de variáveis e completa integração com as IHMs da HI.

Download Gratuito

Editor Ladder

Utilizando Editor Ladder o usuário pode criar o programa de aplicação
que será carregado no controlador, ou seja, é a área de trabalho para o
desenvolvimento do projeto.

Supervisão de Bloco PID

Quando o programa de aplicação carregado no controlador possuir elementos do tipo PID, é possível monitorar os parâmetros através de um painel gráfico.

Dump de Variáveis

Nesta janela é possível que o usuário supervisione e altere os valores correntes das variáveis definidas no programa de aplicação do controlador conectado ao SPDSW.
