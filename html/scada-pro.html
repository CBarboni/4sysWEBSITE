SCADA PRO

Sistema Supervisório SCADA

O HIscada Pro é um pacote de software que disponibiliza um ambiente completo para desenvolvimento, teste e geração de aplicações de supervisão e controle de plantas, processos e máquinas industriais.

Concebido para operar de forma distribuída e utilizando interfaces padronizadas, o HIscada Pro se comunica com servidores de dados OPC, possibilitando o acesso a equipamentos de qualquer fabricante. Possui recursos para construção de telas sinópticas, gerência de alarmes, históricos, relatórios, gerência de usuários, scripts baseados na linguagem LUA e recursos para criação de modelos de processo.
