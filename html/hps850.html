HPS850

HPS850

Sensor de Efeito Hall

O HPS850 é um dispositivo utilizado para detecção de campos magnéticos. Utiliza um transistor de efeito hall como elemento sensor que lhe confere um alto grau de robustez e vida útil, uma vez que não possui partes móveis. É normalmente utilizado em conjunto com um imã permanente como sensor de posição, movimento, contagem de objetos etc.
